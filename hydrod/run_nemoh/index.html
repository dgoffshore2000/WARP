Run Nemoh

Nemoh Execution Modes

Nemoh supports three calculation modes:

​· First-Order Wave Forces (without RAO):
- Outputs: Added mass, radiation damping, and first-order wave forces.
- No MASS parameters required.

​· First-Order Wave Forces + RAO:
- Outputs: Added mass, radiation damping, first-order wave forces and response Amplitude Operator (RAO) results.
- ​MASS parameters must be defined.

​· First-Order + Second-Order Wave Forces (QTF):
- Requires:
MASS parameters.
BVP solution and RAO calculation.
QTF parameter setup.

- Steps:
MASS parameters.
Run BVP solver.
Compute QTF after motion RAO is obtained.

Before hydrodynamic calculations, configure solver parameters. WARP uses ​default recommended values; advanced settings should follow the Nemoh Manual.

MASS Parameter Configuration

Nemoh calculates hydrostatic properties via mesh.exe or hydroCal.exe.

- In WARP, users can ​customize:
Mass/Inertia Matrix
Added Damping
Added Stiffness

Use this page(MASS) to override default values for specialized cases (e.g., additional damping/stiffness).

QTF Parameter Setting

QTF (Quadratic Transfer Function) setup in Nemoh is ​complex.
​Refer to the Nemoh Manual for detailed guidelines.
