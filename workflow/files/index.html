Files

File Structure

​Input Files

Executable Programs

Executable Description
mesh.exe Nemoh model processing & hydrostatic solver
hydroCal.exe Nemoh hydrostatic calculation program
preprocessor.exe Nemoh pre-processing tool
postprocessor.exe Nemoh post-processing tool
QTF_solver.exe Nemoh QTF (Quadratic Transfer Function) solver
solver.exe Nemoh first-order hydrodynamic solver

Input Data Files

File Description
**.dat Nemoh model definition file
ModelFile (no extension) Nemoh model conversion file
mesh.cal, Nemoh.cal Nemoh calculation control files
input_solver.txt Nemoh solver parameter configuration
Mechanics/badd.dat Additional damping matrix
Mechanics/Inertia.dat Mass/inertia matrix
Mechanics/kH.dat Hydrostatic stiffness matrix
Mechanics/kM.dat Additional stiffness matrix

​Output Files

File Description
mesh/KH.dat Hydrostatic stiffness results
mesh/model_info.dat Mesh node/element count and model metadata
mesh/Inertia.dat Inertia matrix results
motion/RAO.dat Response Amplitude Operator (RAO) results
results/CA.dat Added mass coefficients
results/CM.dat Radiation damping coefficients
results/Fe.dat First-order wave excitation forces
results/FKForce.dat Froude-Krylov force results
results/**.tec Tecplot-format hydrodynamic results
results/QTF Second-order wave force (QTF) results

​Process Files

File Description
logfiles/ Runtime log files
**.dg Parameter configuration backup files
summary_*.txt Model summary and hydrostatic key metrics
**.bat Batch files for automated execution via WARP
